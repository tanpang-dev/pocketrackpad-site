A small one-time tool for when PockeTrackPad doesn't appear in your Mac's Bluetooth list.

このツールについて

macOSの「システム設定 > Bluetooth」は、PockeTrackPadのようなBLE(Bluetooth Low Energy)機器を一覧に表示できない場合があります。このヘルパーツールを一度だけ実行すると、MacがPockeTrackPadをBluetooth入力機器として認識できるようになります。インストールは不要で、常駐もしません。

About This Tool

macOS's System Settings > Bluetooth sometimes cannot list BLE (Bluetooth Low Energy) devices like PockeTrackPad. Running this helper tool once lets your Mac recognize PockeTrackPad as a Bluetooth input device. No installation is required, and it does not run in the background afterward.

ダウンロード / Download

PockeTrackPadConnect.zip をダウンロード / Download PockeTrackPadConnect.zip

使い方

  1. ダウンロードしたzipファイルを解凍します。
  2. 「PockeTrackPadConnect.command」をダブルクリックして開きます。
  3. 「開発元が未確認のため開けません」といった警告が表示されたら、「システム設定 > プライバシーとセキュリティ」を開き、下の方にある「このまま開く」を選びます。
  4. もう一度ファイルを開こうとし、表示されたダイアログで「開く」を選びます。
  5. ターミナルが起動し、PockeTrackPadとの接続を待ち受けます。iPhone側でPockeTrackPadアプリを開いていれば、しばらくしてMacがBluetooth機器として認識します。

How to Use

  1. Unzip the downloaded file.
  2. Double-click "PockeTrackPadConnect.command" to open it.
  3. If a warning appears saying it can't be opened because the developer can't be verified, open System Settings > Privacy & Security and choose "Open Anyway" near the bottom.
  4. Try opening the file again, then choose Open in the dialog that appears.
  5. A Terminal window opens and waits for a connection. With the PockeTrackPad app open on your iPhone, your Mac will recognize it as a Bluetooth device shortly after.

マウスが使えないとき

マウスやトラックパッドが使えない場合も、上記と同じ手順をキーボードだけで行えます。

  1. Finderで矢印キーを使って「PockeTrackPadConnect.command」を選び、Command+Oで開こうとします。
  2. 警告が出たら、Command+SpaceでSpotlight検索を開き、「プライバシーとセキュリティ」と入力してReturnキーを押します。
  3. 画面の下の方にある「このまま開く」ボタンをTabキーで選択し、Returnキーで実行します。
  4. もう一度ファイルを開こうとすると起動します。

If You Can't Use a Mouse

If you can't use a mouse or trackpad, you can do the same steps above with the keyboard alone.

  1. In Finder, use the arrow keys to select "PockeTrackPadConnect.command", then press Command-O to open it.
  2. If a warning appears, press Command-Space to open Spotlight, type "Privacy & Security", and press Return.
  3. Use Tab to select the "Open Anyway" button near the bottom of the screen, then press Return to run it.
  4. Try opening the file again and it will launch.
